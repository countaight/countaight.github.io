for...in loop

For loops allow you to execute code an amount of times determined by a variable designated for every element in an enumerable. 

for num in 0..5 do
  puts "Value of num is now #{num}."
end

while/until loop

While loops allow code to execute while a condition is true. Until loops allow code to execute while a condition is false. 

i = 0
while i <= 5 do
  puts "Variable i value is #{i}"
  i += 1
end
					
i = 5
until i > 5 do
  puts "Variable i value is #{i}"
  i += 1
end
whileuntil

loop methods

Ruby also offers methods, also call enumerable methods, that allow code to execute an amount of times or based on the amount of elements in an enumerable, such as an array or hash. 

5.times do
  puts "Running this many times."
end
				
array = [1, 2, 3]
array.each do |number|
  puts "Array has number #{number}."
end
timeseach

for loop

For loops in JavaScript are a bit more involved, but still straight forward. The condition for the loops includes a variable counter assigment, a condition for the variable to assess, and an increment value. 

for (var i = 0; i < 5; i++) {
  console.log("Variable i is", i);
}

for...in loop

For...in loops are similar to Ruby's; they accept a variable for every element in an expression. 

var array = [1, 2, 3, 4, 5]
for (var i in array) {
  console.log("Index", i, "has a value of", array[i]);
}

while loop

While loops in JavaScript execute the code as long as the condition remains true. It's a smart idea to be able to make the condition false, or else you will run into an infinite loop. 

var i = 0
while (i < 5) {
  console.log("Value of i is", i);
  i++;
}
