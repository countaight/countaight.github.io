for...in loop

For loops allow you to execute code an amount of times determined by a variable designated for every element in an enumerable. 

for num in 0..5 do
 puts "Value of num is now #{num}."
end
Value of num is now 0.
Value of num is now 1.
Value of num is now 2.
Value of num is now 3.
Value of num is now 4.
Value of num is now 5.
=> 0..5

while/until loop

While loops allow code to execute while a condition is true. Until loops allow code to execute while a condition is false. 

i = 0
while i <= 5 do
  puts "Variable i value is #{i}."
  i += 1
end
Variable i value is 0.
Variable i value is 1.
Variable i value is 2.
Variable i value is 3.
Variable i value is 4.
Variable i value is 5.
=> nil
i = 0
until i > 5 do
  puts "Variable i value is #{i}"
  i += 1
end
Variable i value is 0.
Variable i value is 1.
Variable i value is 2.
Variable i value is 3.
Variable i value is 4.
Variable i value is 5.
=> nil

loop methods

Ruby also offers methods, also call enumerable methods, that allow code to execute an amount of times or based on the amount of elements in an enumerable, such as an array or hash. 

i = 1
5.times do
  puts "This has run #{i} time(s)."
  i += 1
end
This has run 1 time(s).
This has run 2 time(s).
This has run 3 time(s).
This has run 4 time(s).
This has run 5 time(s).
=> 5
(1..5).each do |number|
  puts "This has run #{number} times."
end
This has run 1 time(s).
This has run 2 time(s).
This has run 3 time(s).
This has run 4 time(s).
This has run 5 time(s).
=> 1..5

for loop

For loops in JavaScript are a bit more involved, but still straight forward. The condition for the loops includes a variable counter assigment, a condition for the variable to assess, and an increment value. 

for (var i = 0; i <= 5; i++) {
  console.log("Variable i is", i);
}
Variable i is 0.
Variable i is 1.
Variable i is 2.
Variable i is 3.
Variable i is 4.
Variable i is 5.
=> undefined

for...in loop

Similar to Ruby's for...in loop; they take a variable which applies an index to each element of an array or object. The variable must also be passed to the array or object in order to access the value. 

var array = [1, 2, 3, 4, 5]
for (var i in array) {
  console.log("Index", i, "has a value of", array[i] + ".");
}
Index has 0 a value of 1.
Index has 1 a value of 2.
Index has 2 a value of 3.
Index has 3 a value of 4.
Index has 4 a value of 5.
=> undefined

while loop

While loops in JavaScript execute the code as long as the condition remains true. It's a smart idea to be able to make the condition false, or else you will run into an infinite loop. 

var i = 1
while (i < 5) {
  console.log("This has run", i, "time(s).");
  i++;
}
This has run 1 time(s).
This has run 2 time(s).
This has run 3 time(s).
This has run 4 time(s).
This has run 5 time(s).
=> undefined
